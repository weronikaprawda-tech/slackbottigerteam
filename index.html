📊 What you’re looking at

This channel is for usage stats — a live Tableau view (Individual Canvas Usage) that shows how Slack canvas activity shows up in our reporting: who’s engaging, how usage breaks down, and trends we care about for adoption and coaching.

The usage piece is about measuring real behaviour — not just that a canvas exists, but how people interact with canvases and related workflows over time, so the Tiger Team and leaders can spot what’s working, where to nudge, and how Slackbot and canvases land in the field.

Slack list: The curated source for what we include lives in this List in Salesforce Slack (sign in if prompted).

Typical things in the grid below include:

  • Breakdowns and filters that slice canvas usage by team, role, or segment (exact fields depend on the workbook).
  • Volume and frequency signals — who’s active, how often, and how that changes week to week.
  • Context for programmes like Prompt Clinics and pilots: are the right people in the habit?

The embedded view is filtered on the Canvas ID dimension via the Tableau Embedding API (same IDs as below), without reloading the whole viz when you change checkboxes. Use in-viz controls for dates and other fields.

    Drill into individual employees

    On the right-hand side of the dashboard, set the grouping parameters (Bucket 1 / 2 / 3) to Mgr Lvl 6 Mgr Lvl 7 and Name to break the table down to individual employee insights. You can also use the Search Name box to jump straight to a person.

    Once you're happy with the layout, use the Tableau toolbar to Save it as a custom view and tick Make it my default — every time you open the dashboard it will load with your preferred grouping already applied.

    Grid view · Individual Canvas Usage

    Same order as this page: Canvas IDs are applied to the viz in A→Z alphabetical order (same lexicographic sort as Tableau’s string dimension). The row order inside the grid is controlled in the published workbook — ask the owner to sort the sheet by Canvas ID ascending so the viz matches this list. If Tableau’s own filter list is hard to scroll (e.g. to reach IDs starting with F09), type F09 above and click Use search in dashboard to narrow the embedded filter to matching IDs only. Open in Tableau still uses URL view filters. If that link misses rows, try TABLEAU_CANVAS_FILTER_USE_VF_PREFIX = true or TABLEAU_CANVAS_FILTER_STYLE = "filter-param". Embedding uses applyFilterAsync on the sheet named TABLEAU_EMBED_FILTER_WORKSHEET (default SLACKBOTPROMPTS on a dashboard). Tableau Embedding API: Filter the view.
